Programming Language Terms

Term Definition
Machine Language The lowest-level programming language, which is directly understood by the computer hardware.
Debugging The process of identifying and removing errors from computer software or hardware.
Assembler A tool that translates assembly language into machine language.
Assembly Language A low-level programming language that uses symbolic code (mnemonic systems) and is closely related to machine language.
Translator A program that converts code written in one programming language into another.
Compiler A tool that translates code from a high-level programming language to a lower-level language to create an executable program.
Interpreter A program that directly executes instructions written in a programming or scripting language without requiring them to be compiled into a machine language program.
Cross-platform Software Software that is designed to work on multiple computer platforms.
Natural Language Any language that has developed naturally in humans through use and repetition without conscious planning or premeditation.
Formal Language A language designed for use in situations in which natural language is unsuitable, such as for mathematics, computer programming, or chemistry.
Machine Independence A property of software that can operate on multiple hardware platforms without modification.
High-level Programming Language Programming languages that are more abstract, easier to program in, and more portable than lower-level languages.
Object-oriented Programming (OOP) A programming paradigm based on the concept of objects, which can contain data and code: data in the form of fields, and code, in the form of procedures.
Scripting Language Languages that automate the execution of tasks that could alternatively be executed one-by-one by a human operator.
Data Types Classifications of data that determine the possible values and operations that can be performed on that data.
Integer A data type that represents some range of mathematical integers.
Float A data type used in computing to represent real numbers with fractional parts.
String A data type used in programming to represent text rather than numbers.
Primitive Data Type The most basic data types available within a programming language, which often include types like integers, floats, and booleans.
Data Structure A particular way of organizing data in a computer so that it can be used efficiently.
Python Collection Data Structures Types in Python that are used to store collections of data, which include list, tuple, set, and dictionary.
List An ordered collection of items which can be of varying data types.
Tuple An immutable ordered collection of items, which means it cannot be modified after it is created.
Set An unordered collection with no duplicate elements.
Dictionary A collection of key-value pairs with each key being unique.
Function A block of organized, reusable code that is used to perform a single, related action.
Local Variable A variable that is declared inside a function and is only accessible within that function.
Global Variable A variable that is declared outside any function and is accessible to all functions within the program.
Source Program The original code written by a programmer in a high-level programming language before it is compiled.
Object Program The output of a compiler after it translates the source program, usually in the form of machine code that the computer can execute.
