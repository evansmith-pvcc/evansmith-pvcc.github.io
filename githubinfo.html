All About Github

    • Q. What is a version control system?
    • A. A software tool that helps to keep track of updates in code.

    • Q. What is Git?
    • A. It is a popular "distributed" version control system that allows users to collaborate effectively

    • Q. What is Github?
    • A. It is a for profit company that offers a cloud-based Git repository. Github's interface makes navigating the repository significantly easier for new developers.

    • Q. What is the difference between Git and Github?
    • A. Git is a specific type of version control system that is distributed as opposed to centralized. Github is a company that uses Git and offers a service that is free for most members.

    • Q. Who started Github and how was it started?
    • A. Github was started by Tom Preston-Werner, Chris Wanstrath, P. J. Hyett and Scott Chacon. The founders were on a mission to democratize coding and created a free online platform.

    • Q. What company owns it now?
    • A. Microsoft Corp is the current owner of GitHub Inc.

    • Q. How much does a Github account cost?
    • A. A Github account is free!

    • Q. What is the Octocat?
    • A. The Octocat was originally an illustration of a half octopus half cat named "Octopuss" that appeared on webpages that didnt work on Github. It now has many variations of it in different clothes and settings and is still in use today.

    Git/Github terms:

    Term Definition/Meaning
    Repository A place where you can store shared code
    Commit Can be thought of as snapshots or milestones, basically saves the project
    Fork A copy of the current code that one can edit without it going back to the base branch
    Push Used to upload from local repository to remote repository
    Pull Request A notification that work is completed on a branch and features are ready to be added to the base branch.
    Workflows How a person or team sets up their repositories to be cosistent and productive
    Issues Flexible items that can be created to plan discuss and track work
    Raw Button Allows the user to see the raw code without any styling
    Blame Button Used to examine a file line by line and see who edited what and when
