Network and Internet Terminology

TERM DEFINITION
Network Classifications
LAN Local Area Network: a network that connects computers and devices in a limited geographical area such as a home, school, or office building.
MAN Metropolitan Area Network: a network that covers a larger geographic area than a LAN and smaller than a WAN, such as a city or town.
WAN Wide Area Network: a network that covers broad geographic areas, such as cities, states, or countries, using leased telecommunication lines.
Network models
bus network A network topology where each of a network's nodes is connected to a common linear backbone that carries messages to all nodes.
star network A network topology in which each network node is individually connected to a central hub, which acts as a conduit to transmit messages.
Protocols
CSMA/CD Ethernet Carrier Sense Multiple Access with Collision Detection: a network protocol for carrier transmission that is used in Ethernet networks to manage data collisions.
CSMA/CA Wifi Carrier Sense Multiple Access with Collision Avoidance: a network protocol that tries to avoid collisions before they happen, used in wireless networks.
Combining networks
repeater A network device that regenerates incoming electrical, wireless, or optical signals to extend the distance over which data can travel.
bridge A device that connects two local area networks (LANs) or two segments of the same LAN that use the same protocol, like Ethernet or Wi-Fi.
switch A networking device that connects devices together on a computer network, using packet switching to receive, process, and forward data to the destination device.
router A networking device that forwards data packets between computer networks, performing traffic directing functions on the Internet.
gateway A network point that acts as an entrance to another network, often used to connect different network architectures, including performing protocol conversions.
Client/server model
P2P: peer-to-peer A decentralized communications model in which each party has the same capabilities and either party can initiate a communication session.
distributed system A network that consists of autonomous computers that are connected using a distribution middleware allowing multiple users to access resources simultaneously.
cloud computing The delivery of different services through the Internet, including data storage, servers, databases, networking, and software.
Internet Addressing
Tier 1 Internet Service Providers (ISPs) that can reach every other network on the Internet without purchasing IP transit or paying settlements.
Tier 2 ISPs that peer with some networks but still purchase IP transit or pay settlements to reach at least some portion of the Internet.
Tier 3 ISPs that must pay to reach the Internet, typically focusing on providing internet access to end-users and businesses.
IP address
ICANN Internet Corporation for Assigned Names and Numbers: A nonprofit organization responsible for coordinating the maintenance and methodologies of several databases related to the namespaces and numerical spaces of the Internet, ensuring the network's stable and secure operation.
DNS Domain Name System: The system by which internet domain names and addresses are tracked and regulated.
TLD: Top-level domain The last segment of the domain name; the part that follows immediately after the "dot" symbol. Examples include .com, .org, and .net.
Country-code TLDs Country-code Top-Level Domains: Two-letter domains established for countries or territories; for example, .uk for the United Kingdom, .jp for Japan.
Internet protocols
FTP File Transfer Protocol: A standard network protocol used for the transfer of computer files between a client and server on a computer network.
HTTP Hypertext Transfer Protocol: The foundation of data communication for the World Wide Web, where data is exchanged between a browser and a server.
SMTP Simple Mail Transfer Protocol: An Internet standard for email transmission across Internet Protocol (IP) networks.
World Wide Web
URL Uniform Resource Locator: A reference to a web resource that specifies its location on a computer network and a mechanism for retrieving it.
client-side activities Actions that occur on the user's computer in a web transaction, such as input validation or running scripts in a browser.
server-side activities Actions that occur on the web server, such as processing form data, managing database interactions, or authorization.
Internet four-layer software
application layer The top layer of the four-layer Internet model that provides networking services directly to applications.
transport layer Provides communication session management between host computers, including the control of data flow.
network layer Responsible for data transfer between two different networks. Handles routing through intermediate routers.
link layer The layer that provides data transfer between adjacent network nodes in a wide area or local area network.
Internet is
packet-switching network A digital networking communications method that groups all transmitted data into suitably sized blocks, called packets, which are transmitted via a medium that may be shared by multiple simultaneous communication sessions.
fault tolerant The ability of a system to continue operation, even if one or more of its components fail.
redundant The duplication of critical components or functions of a system with the intention of increasing reliability of the system.
Cybersecurity
malware Malicious software designed to disrupt, damage, or gain unauthorized access to a computer system.
virus A type of malware that replicates by copying itself to another program, system, or file.
worm A malware computer program that replicates itself in order to spread to other computers, often using a computer network.
Trojan horse A type of malware that is often disguised as legitimate software, used by hackers to gain access to users' systems.
spyware Software that enables a user to obtain covert information about another's computer activities by transmitting data covertly from their hard drive.
phishing The fraudulent attempt to obtain sensitive information or data, such as usernames, passwords, and credit card details, by disguising oneself as a trustworthy entity in an electronic communication.
DDoS attack Distributed Denial of Service: An attempt to make an online service unavailable by overwhelming it with traffic from multiple sources.
Protection
firewall A network security system that monitors and controls incoming and outgoing network traffic based on predetermined security rules.
spam filter A program or system that is used to detect unsolicited and unwanted email and prevent those messages from getting to a user's inbox.
proxy server A server that acts as an intermediary for requests from clients seeking resources from other servers, providing both efficiency and security.
antivirus software Software designed to detect, prevent, and remove malware, including viruses, worms, and Trojan horses.
cryptography The practice and study of techniques for secure communication in the presence of third parties called adversaries.
public key encryption A cryptographic system that uses two keys -- a public key known to everyone and a private or secret key known only to the recipient of the message.
