Operating Systems Terminology

TERM DEFINITION
Operating System An interface between a computer user and computer hardware; manages computer hardware resources and provides common services for computer programs.
Job Queue A list of jobs waiting to be processed by the computer system.
FIFO (First In, First Out) A queueing method where the first task to arrive is the first to be processed.
Interactive Processing A method of processing that allows a user to interact with a program while it is running.
Time-Sharing A technology that allows multiple users to use a computer at the same time, by allocating a computer's resources in time slices.
Multiprogramming A method of running several programs on a single processor by interleaving the execution of each program in time segments.
Application Software Software designed to help the user to perform singular or multiple related specific tasks.
System Software Software designed to provide a platform for other software. Examples include operating systems, device drivers, and utilities.
Utility Software System software designed to help analyze, configure, optimize or maintain a computer.
Graphical User Interface (GUI) An interface that allows users to interact with electronic devices through graphical icons and visual indicators.
Shell A user interface for access to an operating system's services. It can be command-line-based or graphical.
Kernel The core component of an operating system, managing system resources and the communication between hardware and software components.
Device Driver Software that tells the operating system and other software how to communicate with a piece of hardware.
File Manager A software that provides a user interface to manage files and folders.
Memory Manager A function of an operating system that handles or manages primary memory and moves processes back and forth between main memory and disk during execution.
Virtual Memory A memory management technique that provides an "idealized abstraction of the storage resources that are actually available on a given machine."
Boot Strapping The process of starting up a computer from a halted or powered-down condition.
ROM Read-Only Memory, a type of non-volatile storage that is used in computers and other electronic devices to store firmware.
Deadlock A situation where two or more processes are unable to proceed because each is waiting for the other to release resources.
Spooling A process in which data is temporarily gathered to be used and executed by a device, program or the system.
Super User (Or Administrator) A user with the highest level of access to the computer's operating system, able to perform all administrative tasks.
Multi-Factor Authentication A security system that requires more than one method of authentication from independent categories of credentials to verify the user's identity for a login or other transaction.
